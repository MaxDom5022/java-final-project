# RBC Java Game hub
The Java Game hub is a Multi java program consisting of five games

# games
1. Guess the Number - A number guessing game where the program picks a random number 1-100 for the user to guess.
2. Rock Paper scissors - a classic game where the user picks rock, paper,or scissors and the computer makes a random choice.
3. Trivia Game - multiple choice trivia game with 6 questions
4. Word Scramble - The computer Randomly selects a word and scramblses the letters the user has to guess the correct word.
5. High Low card game - The Program Draws a Cars value from 1 to 13 and then the user will guees wheather the next card is higher or lower.

# project Structure
game.java - the parent class that extends to all the games(Subclasses), Contains a start(), play(), end()
MainMenu.java - the main class that launches the games
guessTheNumber.java - Guess the number Game 
rockPaperScissors.java - Rock PAper Scissors game class
trivia.Java - trivia game class
wordScramble.java - Word Scramble game class
highLowCardGame.java - High Low Card game class

# How to Run the RBC Java Game Hub
1. Open the project in the IDE of your Choice
2. Make Sure all java files are in the same Folder
3. Once All files/classes are opened in your IDE of choice, Begin Playing throgh MainMenu.java
